# Petal Budget Tracker

A personal finance web app for tracking expenses, visualising spending by category, and monitoring savings goals.

## Features

- Add, edit, and delete expense entries
- Filter expenses by category
- Category split chart for visual spending breakdown
- Savings card to track progress toward goals
- Authentication page

## Tech Stack

React · JavaScript · Vite · CSS

## Getting Started

```bash
npm install
npm run dev
```
